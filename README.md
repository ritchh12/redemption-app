# Redemption App & Log Analysis

A comprehensive analysis and management system for code redemption tracking and log analysis.

## Project Contents

- **code_redemption_analysis.ipynb** - Jupyter notebook with detailed redemption code analysis
- **professional_sql_analysis.sql** - SQL queries for data analysis
- **attempts_data.csv** - Dataset containing redemption attempts
- **users.csv** - User data
- **flagged_users.csv** - Flagged or suspicious user accounts
- **Report.pbix** - Power BI report for visualization

## Features

- Code redemption tracking and analytics
- User activity monitoring
- Log analysis and reporting
- Data visualization

## Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- Required Python packages (see requirements.txt)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/redemption-app.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the analysis:
```bash
jupyter notebook code_redemption_analysis.ipynb
```
