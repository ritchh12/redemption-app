# 💰 Redemption App & Fraud Analysis System

A full-stack **Code Redemption System + Data Analysis Project** designed to track user activity, detect fraud patterns, and generate actionable insights using SQL, Python, and Power BI.

## 📌 Project Overview

This project simulates a real-world **code redemption platform** and performs end-to-end analysis on user behavior and fraudulent activities.

It includes:

* Backend logic for redemption & fraud detection
* SQL-based analytical queries
* Python-based data analysis
* Power BI dashboard for visualization

---

## 🎯 Objectives

* Analyze user redemption behavior
* Identify failure patterns
* Detect fraudulent users
* Improve system efficiency

---

## 🧠 Key Concepts

* Fraud Detection Logic
* User Behavior Analysis
* KPI Monitoring
* Data Visualization

💡 **Insight (meaningful understanding from data)**

## 📂 Project Structure

* **code_redemption_analysis.ipynb** → Python analysis notebook
* **professional_sql_analysis.sql** → SQL-based analysis queries
* **attempts_data.csv** → Redemption attempt logs
* **users.csv** → User data
* **flagged_users.csv** → Suspicious users
* **Report.pbix** → Power BI dashboard

## 🧠 Key Insights

* Majority of attempts are successful → system is efficient
* Invalid and reused codes are main failure reasons
* A small group of users contributes to most failed attempts
* Fraud users show repeated rapid attempts
* Peak activity occurs during evening hours
